# OpenMP
First steps in learning OpenMP
В первой задаче массив заполняется значениями синуса в промежутке от 0 до pi/2, сравнивается время выполнение последовательной прогрыммы и распаралелленой.
Во второй задаче считается интеграл методом прямоугольников.
В третьей задаче ищутся количество простых чисел в промежутке от 1 до n.
В четвертой задаче имеется массив точек, необходимо вычислить максимальное расстояние между двумя произвольными точками
В пятой задаче расчитывается следующий ход игры жизнь
